    Create coordinator.py following RESEARCH Pattern 1. Key implementation constraints:

    Imports: from __future__ import annotations, asyncio, logging, datetime.timedelta, typing.TypedDict,
      homeassistant.core.HomeAssistant, homeassistant.helpers.aiohttp_client.async_get_clientsession,
      homeassistant.helpers.update_coordinator.{DataUpdateCoordinator, UpdateFailed},
      homeassistant.exceptions.ConfigEntryAuthFailed, .const.{DOMAIN, DEFAULT_SCAN_INTERVAL}

    Use asyncio.timeout(10) — NOT async_timeout.timeout (banned per CLAUDE.md §What NOT to Use).

    _async_update_data() error classification (CRITICAL — per D-03, D-04, D-05, RESEARCH Pitfall 3):
      - Any exception from session.get() call → raise UpdateFailed (network/timeout errors)
      - resp.status in (401, 403) → raise ConfigEntryAuthFailed (do NOT raise UpdateFailed for auth errors)
      - resp.status != 200 → raise UpdateFailed with "Unexpected HTTP {resp.status}"
      - resp.json() parse error → raise UpdateFailed with "Invalid JSON from Gatus: {err}" (log the error)
      - NEVER raise ConfigEntryNotReady from _async_update_data; the auto-conversion from UpdateFailed
        to ConfigEntryNotReady happens in async_config_entry_first_refresh() in __init__.py

    _async_update_data() returns a fresh dict built from the current API response.
    Endpoints absent from the API response are simply not included in the returned dict.
    DataUpdateCoordinator replaces coordinator.data with the returned dict on each successful call.
    This implements D-02 (disappearing endpoint key removed) with no extra code — the fresh dict
    naturally excludes stale keys.

    GatusEndpoint duration_ms: latest result "duration" nanoseconds // 1_000_000 (integer division).
    GatusEndpoint condition_results: raw conditionResults list from the latest result dict.
    If endpoint has no results: default all numeric fields to 0, success to False, timestamp to "".

    API key: if self._api_key is truthy, headers["Authorization"] = f"Bearer {self._api_key}".
    NEVER log self._api_key or the Authorization header value (ASVS V2).

    Constructor: strip trailing slash from url with url.rstrip("/").
    Pass update_interval=timedelta(seconds=scan_interval) to super().__init__.

    conftest.py: Add enable_custom_integrations fixture (required by pytest-homeassistant-custom-component).
    Also ensure custom_components/ is on sys.path so pytest can import the integration.
