Create `.github/workflows/validate.yml` with two jobs:

**Job 1: hacs** — HACS validation
- Uses: `hacs/action@main`
- env: `GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}`
- with: `category: integration`, `ignore: brands` set to `false` (do NOT ignore brands — icon.png will be provided)

**Job 2: hassfest** — HA integration validator
- Uses: `home-assistant/actions/hassfest@master`

**Triggers:**
- `push` (all branches)
- `pull_request`
- `schedule`: cron `"0 0 * * *"` (daily midnight UTC)
- `workflow_dispatch`

Full YAML:

```yaml
name: Validate

on:
  push:
  pull_request:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:

jobs:
  hacs:
    name: HACS validation
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: hacs/action@main
        with:
          category: integration
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}

  hassfest:
    name: Hassfest validation
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: home-assistant/actions/hassfest@master
```

Create the `.github/workflows/` directory if it does not exist, then write the file.
